## Getting started

You will need Docker, Docker-Compose and Git.

Run the commands below on your favorite terminal:
 
```bash
git clone git@github.com:ravanscafi/mongodb-workshop-base.git
cd mongodb-workshop-base
docker-compose up -d
docker-compose exec php composer setup
```

Now go ahead and access [localhost](http://localhost).

If you are looking for the slides, check [ravanscafi/mongodb-workshop](https://github.com/ravanscafi/mongodb-workshop).
